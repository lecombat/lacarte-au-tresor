# Carte aux tresors
### Guidez les aventuriers en quête de trésors !

### Build
```mvn clean install```

### Run
Le projet est builder avec open JDK 15

```java -jar targer/lacarte-au-tresor-1.0-SNAPSHOT.jar <cheminAbsoluFichierInitCarte> <cheminAbsoluFichierSauvegardeCarte>```
